├─ ML_MODEL_PATH: "/app/models/xgb_anomaly_model.pkl"
├─ VPN_DETECTION_THRESHOLD: "0.8" (IPQS confidence)
├─ PROXY_DETECTION_THRESHOLD: "0.8" (IPQS confidence)
├─ BOT_SCORE_THRESHOLD: "0.7" (UA analysis)
├─ IMPOSSIBLE_TRAVEL_SPEED_KMH: "900" (max realistic speed)
├─ DORMANCY_DAYS: "30" (account inactivity)
├─ BLOCKED_COUNTRIES: "IR,KP,SY" (fully blocked)
└─ RESTRICTED_COUNTRIES: "CN,RU,KR" (flagged)

API Rate Limits:
- IPQS: 5000 req/month (free tier cache recommended)
- HIBP: 1 req/120s (built-in rate limiting)
- GreyNoise: Depends on enterprise plan
- SIEM webhook: Depends on SIEM capacity

Database Indexes:
- idx_user_id: User lookup optimization
- idx_ip: IP address lookup
- idx_risk_score: Risk analysis queries
- idx_anomaly_score: ML-based filtering

TTL & Retention:
- Daily partitioning by event_time
- 90-day TTL (old data deleted automatically)
- Can be modified in ALTER TABLE statement
